# vue-gin-blog
vue+gin全栈博客项目

 这是一个用来练手的项目，vue3+go的博客系统，跟着b站视频做的，老师讲的很不错，项目源地址
 https://github.com/wejectchen/Ginblog.git
 不过老师用的是vue2，这个项目还有很多瑕疵，我原本以为这是一个那种博客网站系统，做到后面个人博客网站，所以感觉后台管理系统中的用户管理
 和一些权限设置没什么必要。
 ### 项目使用
 ##### 1 在根目录执行`go mod tidy` 下载go的一些包
 ##### 2 在web/admin和web/view中分别执行`npm i`，下载前端依赖
 ##### 3 在根目录执行`go run main.go`启动后端
 ##### 4 在web/admin和web/view分别执行`npm run dev`，启动前后台的项目
